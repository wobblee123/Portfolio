Bread Beater

A simple clicker game built with Python and Tkinter library.

Overview

Bread Beater is a beginner-friendly incremental game where you:

- Click to generate bread
- Purchase upgrades to boost your bread production
- Watch numbers increase over time

Features

- Manual clicking system
- Multiple upgradeable buildings:

  * Loaf
  * Bakerer
  * Dough Machine
  * Wind Mill
* Scaling cost system for upgrades
* Real-time UI updates using Tkinter

How to Run

1. Make sure you have Python installed (3.x recommended)
2. Run the script:

   python bread_beater.py (in cmd)

3. Start clicking and building your bread production

Built With

- Python
- Tkinter (standard GUI library)

Purpose

This project was created for learning purposes, focusing on:

- GUI development with Tkinter
- Basic game loops and mechanics
- State management in Python

Known Issues

- Prices do not visually update on buttons after purchase
- No saving system
- Balancing may not be fully refined

This project is intended as a simple learning exercise and can be expanded or modified further.
